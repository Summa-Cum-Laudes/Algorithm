# 00. 10951번:A+B-4
## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력
입력은 여러 개의 테스트 케이스로 이루어져 있다.

각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)
## 의사코드
반복한다
 시험한다
 A,B를 입력한다
 출력한다 A+B
 에러가걸릴시 멈춘다.

```
반복한다.
 시험
  A,B를 입력한다.
  출력한다 A+B
  에러걸릴시 =멈춘다.
```

## 풀이
```python
while True:
  try:
    A,B=map(int,input().split())
    print(A+B)
  except:
    break
```
