# 00. 2439번 별찍기-2
## 문제
첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

하지만, 오른쪽을 기준으로 정렬한 별(예제 참고)을 출력하시오.
## 의사코드
star입력
반복입력 1씩올라가게 star+1
프린트 공백 +스타 << 공백이 앞을 채우고 뒤에 스타가 오게
```
star입력
반복입력 (1,스타 +1)
 프린트 (공백)*(스타-i)+("*")*i 
```

## 풀이
```python
star=int(input())
for i in range(1, star + 1):
  print(" " * (star - i) + "*" * i)
```
